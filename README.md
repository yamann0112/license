# Bot Lisans Yönetim Paneli

Bu dosya üzerinden bot kullanıcılarının lisans sürelerini ve HWID (Donanım Kimliği) kilitlerini yönetebilirsiniz.

## Lisans Formatı
Her satır bir lisansı temsil eder ve format şu şekildedir:
`ANAHTAR-YIL-AY-GÜN-HWID`

* **Süreli Lisans:** `KEY123-2026-12-31-HWID_BURAYA`
* **Süresiz Lisans:** `KEY123-PERMANENT-HWID_BURAYA`
* **Her PC'de Çalışan Key:** `KEY123-2026-12-31-ANY`
